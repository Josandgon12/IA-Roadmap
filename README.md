# IA Roadmap

Este repositorio contiene los proyectos de estudio y desarrollo relacionados con Inteligencia Artificial.

## Proyectos 🚀

*   **Agente:** Proof of Concept (PoC) de una agencia de software virtual implementada con el framework CrewAI.
*   **Chatbot:** PoC de un chatbot de e-commerce utilizando la SDK de Google GenAI y Function Calling.
*   **Local:** Entorno e implementación de RAG orientado a ejecución local.
*   **MCP:** Código de prueba y exploración del Model Context Protocol (implementación simple de cliente y de servidor desde cero).
*   **RAG:** Sistema básico de Retrieval-Augmented Generation.
